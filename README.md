## Opinions

### About AI

A lot of people are quite binary on this topic. Either you hate AI or you love it and think it will be the future.

I think it's a useful tool. If you want to learn coding, it is a great tool when you use it correctly.

AI shouldn't do all of the work by itself. It should assist you with boring tasks, like a Roomba.

A good example is motion capture. There were movies that only used motion capture/used motion capture as the main method of animation. Lot's of studios started to use it/misuse it. People started to complain. And then it found it's place. Now, it's it's own carrer path in animation.
The same thing could happen with AI. Maybe. I can't look into the future.
